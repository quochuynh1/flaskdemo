About This App

This web app allows users to search Wikipedia and view article summaries using the Wikipedia API and Flask.

Demonstrates:

  • Jinja template inheritance
  • HTML forms and POST requests
  • API integration

Created as part of learning Flask for cp1404 prac_10 at JCU.
